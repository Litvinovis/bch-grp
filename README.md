# БЧ-ГРП — Discord RPG Бот

**БЧ-ГРП** (BCH-GRP) — многопользовательская текстовая RPG, работающая как Discord-бот. Написана на Java 25 с PostgreSQL и JDA 6. Создана по мотивам легендарной ЧБ-РПГ.

## Игровые системы

### Персонаж и прогрессия

Персонаж имеет характеристики: ❤️ HP, ⚔️ Сила, 🛡️ Броня, 🍀 Удача, ⭐ Репутация, 💰 Монеты. С каждым уровнем растут HP и сила. На 100-м уровне доступен `+престиж` — сброс уровня с постоянным бонусом +5 ко всем статам.

**Звания:** Новичок → Искатель → Герой → Легенда

### Классы и навыки

Класс выбирается с 5-го уровня (`+класс`), второй класс открывается с 50-го (`+второй класс`).

| Класс | Бонус | Особенность в бою |
|---|---|---|
| ВОИН | +HP, +Броня | Регенерация 5% HP в раунд |
| МАГ | +Сила | Первый удар без ответки; телепорт (`+умение телепорт`) |
| РАЗБОЙНИК | +Удача | Контратака при уклонении |
| ПАЛАДИН | +Броня | Лечит союзников в рейдах; `+умение исцеление` |
| НЕКРОМАНТ | — | 30% шанс поднять убитого моба союзником |
| СЛЕДОПЫТ | +Ресурсы | Видит мобов в локации при входе; +10% к добыче |

**Дерево навыков** (`+скиллы`): каждые 5 уровней — 1 очко навыка. У каждого класса 3 навыка, каждый прокачивается до 3-го уровня. Вложить: `+вложить <навык>`. Активные умения использовать: `+умение <навык>`.

### Боевая система

Каждый раунд: уклонение (зависит от удачи), блок (зависит от брони), контратака разбойника при уклонении, регенерация воина. Маг бьёт первым без ответки в 1-м раунде. Чистая победа = +5 репутации. `+последний бой` — детали последней схватки.

### Питомцы и маунты

Яйца питомцев выпадают с боссов (2%). Легендарный питомец — 2% с рейд-босса. Команды: `+питомец` (инфо/получить), `+кормить <предмет>` (восстановить голод). После 10 боёв питомец эволюционирует (бонус ×2).

| Питомец | Бонус | Спецатака (после раунда 3) |
|---|---|---|
| ВОЛК | +3/+6 к силе | 50% силы игрока — прямым ударом |
| ЛИСА | +3/+6 к удаче | Крадёт 20 монет у противника |
| КОТ | Реген 5/10 HP в раунд | Исцеляет игрока на 30/60 HP |
| ВОРОН | Видит статы врага / +15% уклонение | — |
| ЛЕГЕНДАРНЫЙ | +5/+10 ко всем статам | Всё вышеперечисленное |

Маунт (`маунт ветра` из редкого дропа) сокращает кулдаун перемещения вдвое. `+гонки маунтов` / `+гонки маунтов старт` — еженедельное событие.

### Профессии и крафт

Выбор профессии: `+профессия выбрать <профессия>`. Сбор ресурсов: `+добыть` (кулдаун 30 мин). Крафт: `+создать <рецепт>`. Рецепты: `+рецепты`. Рынок: `+биржа ресурсов`.

| Профессия | Ресурс | Крафтит |
|---|---|---|
| Кузнец | Руда (качалочка, деградач) | Кованый меч (+5 силы) |
| Алхимик | Травы (болото, чебеграм) | Зелье алхимика (+50 HP, 60 мин) |
| Повар | Древесина (для-флуда, дорогой-дневник) | Блюда (баффы 2 часа вместо 30 мин) |
| Ювелир | Камень (клоунская-братва, политота) | Кольца и амулеты (+сила/удача) |

Каждые 10 крафтов — уровень профессии. На 5-м уровне — бонус к стату, на 10-м — мастер-рецепт.

### Кланы и территории

**Клан** (до 5 игроков): `+новый клан`, `+вступить в клан`, `+принять заявки`.

**Расширенное управление:**
- `+клан банк` / `+клан положить` / `+клан снять` — общая казна (снимать — только лидер)
- `+клан улучшения` / `+клан купить <улучшение>` — 3 апгрейда: дроп (500 монет), опыт (750), броня (1000)
- `+клан повысить @игрок` — роли: рядовой / ветеран / офицер / лидер
- `+клан выгнать @игрок`, `+клан база <локация>`

**Территории:**
- `+захватить <локация>` — 3+ члена клана в локации + бой со стражником
- Владелец территории собирает 5% налог со всех транзакций в ней
- `+осада <клан>` — 7-дневная война за территорию. `+осада статус`
- `+крепость строить` (1000 монет), `+крепость улучшить <кузня/таверна/башня>`
  - Кузня: +5% к дропу для клана (500 монет)
  - Таверна: азартные игры без комиссии (300 монет)
  - Башня: +10 брони при защите (700 монет)
- `+альянс <клан>` — союз, +10% урона в совместных рейдах
- `+карта кланов` — кто что контролирует

### Мировые события

| Событие | Периодичность | Команда |
|---|---|---|
| Мировой босс (5 видов) | Каждые 3 дня | `+мировой босс` |
| Нашествие (10 волн NPC) | Еженедельно | `+нашествие` |
| Экономический кризис (цены ×2, 24ч) | Случайно (10%/день) | `+кризис статус` |
| Сезонные предметы в магазине | По месяцу | `+сезон` |
| Серверный турнир (8 чел.) | Ежемесячно | `+турнир` |

**Мировые боссы:** Тёмный Страж (5000 HP), Великий Дракон (7000 HP), Повелитель Хаоса (6000 HP), Ледяной Колосс (8000 HP), Кровавый Берсерк (4500 HP / сила 50). Каждый атакует в ответ и оставляет уникальный лут.

### Таверна и экономика

- `+рулетка <ставка> <красный/черный/0-36>` — цвет платит 1:1, число 35:1; в джекпот копится 5% ставок
- `+скачки` / `+поставить <конь> <ставка>` — 5 коней, шансы зависят от удачи
- `+покер @игрок <ставка>` — карточная дуэль 5-карточный покер
- `+кредит <сумма>` (макс. 500 монет, 5%/день) / `+погасить`
- `+биржа` — продавать дроп с мобов, цена меняется ±20% ежедневно
- `+торговец` — 3 случайных предмета, обновляется раз в день
- Богатство > 10 000 монет → налог 1%/день

### Арена и соревнования

- `+арена` — PvP с ELO-рейтингом (начальный рейтинг 1000). `+арена топ` — топ-10
- `+лига` — твой дивизион: Бронза (<1200) / Серебро (<1500) / Золото (<1800) / Платина (1800+)
- `+арена 3v3` — командный бой три на три
- `+выживание` — все игроки в локации, PvP до последнего, пул ставок победителю
- `+турнир` — регистрация; 8 участников → автоматическая олимпийская сетка; победитель +1000 монет
- `+чемпион` — кто сегодняшний чемпион. `+вызвать чемпиона` — победа даёт +500 монет + достижение

### Фракции и социальные механики

**Репутация фракций** (`+фракции`): ТОРГОВЦЫ (за покупки), МАГИ (за квесты), ВОИНЫ (за убийства мобов). При 100+ репутации: скидка 5% / бонус к квестам / +5 урона.

**Дневник**: `+дневник [запись]` — личные записи в БД (макс. 200 символов, 20 записей). `+дневник` — читать последние 5.

**Бонты**: `+бонт @игрок <сумма>` — поставить награду за голову. `+бонты` — список активных. Награда выплачивается автоматически при PvP-убийстве цели.

**Лор мира**: `+лор` — 10 фрагментов истории, открываются по мере прохождения локаций.

**Скрытые квесты**: 10 Easter Egg фраз — напиши правильную, получи бонус.

**Доска почёта**: `+доска` — еженедельный топ PvP / охотников на мобов / богатейших.

## Конфигурация

Создайте `src/main/resources/application.yml`:

```yaml
discord:
  allowed-channel-ids:
    - "CHANNEL_ID_1"
    - "CHANNEL_ID_2"
  admin-ids:
    - "ADMIN_USER_ID"

db:
  url: "jdbc:postgresql://127.0.0.1:5432/bchgrp"
  username: "bchgrp"
  # password: через переменную окружения DB_PASSWORD
```

Токен Discord передаётся через переменную окружения:

```bash
export BCHGRP_DISCORD_TOKEN="ваш_токен"
```

Конфиги игрового контента (редактируются без перекомпиляции):
- `src/main/resources/locations.yml` — все локации с путями, боссами и PvP-флагами
- `src/main/resources/items.yml` — все предметы с характеристиками и ценами
- `src/main/resources/bosses.yml` — все боссы с HP, силой и лутом

Применить изменения конфигов без перезапуска: `+admin reload`

## Локальный запуск

### Требования

- JDK 25+
- Maven 3.8+
- PostgreSQL 16+

### База данных

```sql
CREATE DATABASE bchgrp;
CREATE USER bchgrp WITH PASSWORD 'пароль';
```

## CI/CD и деплой

Репозиторий использует GitHub Actions с self-hosted runner.

**Пайплайн** (push в `master`):
1. `mvn package -B` — сборка + все тесты
2. Fat-jar копируется в `/opt/BCHGRP/bchgrp.jar`
3. `systemctl restart bot-bchgrp`
4. Smoke-test (30 сек) — при падении автооткат на `bchgrp.jar.backup.*`
5. Уведомление в Telegram

**GitHub Secrets:**

| Secret | Описание |
|---|---|
| `TELEGRAM_BOT_TOKEN` | Токен бота для уведомлений |
| `TELEGRAM_CHAT_ID` | ID чата |

Токен Discord хранится в `.env` на сервере и загружается через `EnvironmentFile` в systemd-сервисе.

**Сервис:** `systemctl status bot-bchgrp`

**Метрики:** `http://localhost:9090/metrics` (Prometheus-формат)
